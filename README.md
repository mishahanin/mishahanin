### What I run on

**HEADING OS** — the system I built to run 31C from the terminal, on top of Claude Code.
It's what I wished existed: an executive's working environment that learns how I think and never leaks what it learns. What it does, day to day:

 - **Relationships & pipeline** — CRM, deal intelligence, and a real read on every counterpart before I walk into the room.
 - **Communications** — email and Telegram drafted in my own voice. Nothing leaves without me clicking send; the system proposes, I decide.
 - **A knowledge brain** — everything I read and every call I make, kept and recalled with sources, so it sharpens the longer I use it.
 - **Built for a fleet** — each executive gets a private workspace; shared knowledge flows down, private data never flows out.

One rule sits under all of it: my data stays on infrastructure I control, never a third-party cloud. The same sovereignty principle behind ODUN.ONE, turned inward on how I work.
